# Аналитика баковского приложения
## Описание
Приложение для анализа финансовых транзакций из Excel файлов.

## Возможности

- Полная аналитика (статистика по картам, курсы валют, цены акций)
- Поиск транзакций по описанию
- Отчёт по категории за последние 3 месяца

## Установка и запуск

1. Клонируйте репозиторий:
```
https://github.com/yulechka944992/Kurs_1
```

2. Установите зависимости:
```
poetry install
```
3. Подключите виртуальное окружение
```
poetry shell
```

## Запустите приложение
```
python src/main.py
```

## Настройка
1. Создайте файл .env в корне проекта:

EXCHANGE_RATE_API_KEY=ваш_ключ_api
ALPHA_VANTAGE_API_KEY=ваш_ключ_api

2. Создайте файл user_settings.json (необязательно):

json

{
    "user_currencies": ["USD", "EUR"],
    "user_stocks": ["AAPL", "AMZN", "GOOGL", "MSFT", "TSLA"]
}

3. Поместите Excel файл с транзакциями в папку data/operations.xlsx

Требования к Excel файлу: столбцы должны называться:

    Категория — категория транзакции

    Сумма операции — сумма (число или строка с запятой)

    Дата операции — дата в формате 01.01.2024 10:30:00

## Логирование

Логи сохраняются в папку logs/main.log
